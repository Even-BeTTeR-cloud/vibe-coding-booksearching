# 코알누 스터디 - 독서여행 (Book Search)

네이버 도서 검색 API를 활용한 교보문고 스타일의 프리미엄 도서 검색 및 포스터 웹페이지입니다.

## 기능
- **교보문고 스타일 디자인**: 프리미엄한 감성의 UI/UX
- **마스터피스 포스터**: 가장 유사한 검색 결과를 대형 포스터 카드로 강조
- **검색 결과 리스트**: 상세 정보 및 구매 링크 제공
- **반응형 웹**: 모바일에서도 최적화된 화면

## 시작하기
1. 의존성 설치: `npm install`
2. 서버 실행: `npm start`
3. 접속: `http://localhost:3000`

## 기술 스택
- Backend: Node.js, Express, Axios, xml2js
- Frontend: Vanilla JS, CSS (Kyobo Design System Inspired)
- API: Naver Search (Book)
